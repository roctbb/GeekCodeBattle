Необходимо разработать систему для проведения батлов по программированию для школьников. В каждом сражении участвует N
школьников и некоторый пул задач, в процессе соревнования выбирается два свободных школьника и объединяются в комнату, в
которой им выпадает задача, кто решил задачу быстрее, тот и победил.

Задачи могут быть на разных языках программирования, оцениваются тестами или нейронкой. Для проверки можно использовать
GeekPaste API (см ../GeekPasteV2 и ../GeekExam). В некоторых задачах может быть критично качество кода.

Для авторизации нужно использовать GeekClass JWT (опять же смотри ../GeekPasteV2 и ../GeekExam).

Проект должен разворачиваться в Docker контейнере, с использованием PostgreSQL для хранения данных о пользователях,
задачах и сражениях.

Код можно писать на Python + Flask + Flask-SQLAlchemy + Flask-Migrate. На фронтенде - Vue 3, для реального времени можно
использовать веб-сокеты.

В системе храниться пул задач и набор сражений, битву можно запускать и останавливать. Учителя и администраторы могут
редактировать как задачи, так и битвы. Нужна возможность ипорта и экспорта задач в json. Для оформления задач надо
использовать markdown.

В каждой комнате определяется победитель раунда, который получает бонус за победу. Можно ввести понуся за цепочку побед
и красивые анимации. Проигравший получает баллы за частичное решение. За победу после цепи поражений тоже можно давать
бонус.

При прочих равных стоит подбирать соперника похожего уровня. Для этого можно вводить небольшую задержку перед началом
раунда, на случай, если появятся новые кандидаты.
